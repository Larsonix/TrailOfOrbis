## Potential NEW Biomes (themed around unused skeleton types)

### "Frozen Crypts" — Pure Frost Undead Biome
- All 8 Skeleton_Frost roles
- Golem_Crystal_Frost as boss or heavy
- Zombie_Frost
- Spirit_Frost
- Would need ice/underground terrain

### "Sand Tombs" — Pure Sand Undead Biome
- All 8 Skeleton_Sand roles + 4 Dungeon variants
- Desert terrain but underground (pyramid interior?)
- Boss: Skeleton_Sand_Archmage?

### "Burning Graves" — Pure Burnt Undead Biome
- All 8 Skeleton_Burnt roles (Knight, Gunner, Wizard, Praetorian, Archer, Alchemist, Lancer, Soldier)
- Zombie_Burnt
- Fire/lava terrain
- Already partially in Desert + Volcano — would need differentiation

### "Radiant Forge" — Incandescent Undead Biome
- Skeleton_Incandescent_Fighter, Footman, Mage, Head
- Visually red/volcanic — confirmed working April 20
- Currently added to Volcano. Could be standalone if more variety found.
