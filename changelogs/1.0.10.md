## Trail of Orbis v1.0.10 - Patch Notes

This one's mostly about feel, polishing, and fixing some backend bugs.
The portal got a full cosmetic pass, I'm really happy, still some clear polish and finitions needed.
Attack speed got a full rework, portals look insane now, with animated images and all that good stuff, and a bunch of things that froze or stuttered don't or shouldn't anymore.

---

### Attack Speed
- **Full Rework.**
- **Combo System.** Attacks speed up the longer you fight. Hit 1 is normal, hit 2 is faster, hit 3 is peak, and it stays there. Resets if you stop attacking
- **Animation Matches Server.** The "swing is done but damage hasn't landed" feel is gone, animations can't outrun the server now
- **6 New Speed Stats.** Cooldown recovery, combo speed, cast speed, projectile speed. Now fully working and polished
- **Hexcode Speed Buffs.** Hex mods can now push temporary speed overrides (haste, slow)

### Weapons
- **Two-Handed Weapons Got a Buff.** All modifier values on two-handed weapons are now 50% stronger than one-handed. You loose the offhand slot, so the weapon itself now makes up for it properly
**Existing items keep their old values, stone rerolls or new drops should have the new numbers.**
- **Mace is Two-Handed Now.** Was getting all the penalties of 2H (no offhand) without the bonuses. Gets the full 1.5x treatment now

### Portals
- **Full Visual Overhaul.** Activated portals now show biome-colored particles, rarity glow, animated biome preview through the portal frame. You can see what you're getting into before entering
- **Rarity Particle Aura.** Higher rarity maps glow more, so you can tell at a glance if someone opened something good - now sure it shows enough
- **Stuck Gateways Fixed.** Realms that were activated but never entered would lock the gateway forever. They auto-close after 120s now
- **Gateways No Longer Spawn in Instances.** Forgotten Temple was generating gateways inside realm worlds

### Compass
- **Height Indicators.** Mob markers show up/down arrows when mobs are above or below you
- **Distance Scaling.** Markers get smaller and fade out the further away the mob is.

### Loot
- **Rarity Drop Particles.** Now show properly
- **Stone Pickup Notifications.** Picking up a stone shows a rarity-colored toast like gear does
- **Loot Filter UI Reworked.** The modifier conditions are fully interactive now, no more `/lf` commands needed. Category tabs, stat pickers, value ranges, all in the UI
- **Stacking Pickups Work.** Well, not perfect yet
- **Chests Don't Reroll on Restart.** Server restart was re-generating chest contents from vanilla drop tables, replacing anything you left in there, I'm so sorry...

### Stones
- **Fortune's Compass Bulk Apply.** Sneak-click to apply all your compasses to a map at once until they run out or the +20% cap is hit
- **All Changes Show in Toast.** Stone modification notification now shows every stat line, not just the first one
- **Alterverse Shard Crash Fixed.** Redistributing between prefix and suffix could exceed the rarity's max modifier count
- **Escape Closes the Picker.** Stone picker and skill node detail both respond to Escape and their close buttons properly now

### Combat
- **Thorns Kills Count.** Thorns damage killing a mob now gives XP, drops loot, counts for realm progress, triggers on-kill effects etc...
- **Tamed Horses Survive World Transitions.** Mounts were dying immediately after teleporting
- **Red Screen Toned Down.** DOT damage no longer triggers the red flash. Threshold raised to 15% of max HP and cooldown increased so multi-mob fights aren't just your character pulsing to death

### Hexcode Compatibility
- **Glyph HUD Works now !**

### UI & HUDs
- **Skill Point HUD Survives the Sanctum.** Entering the skill sanctum was destroying the skill point HUD permanently
- **Signature Ability HUD Restored.** The weapon ability indicator was missing for RPG weapons
- **Chat Item Links.** Shift-clicking items into chat now shows rarity-colored item names in brackets instead of raw `<Item:id>` tags

### Stability
- **No More Stutter on Mob Kills.** Loot generation was blocking the main thread for up to 200ms per item. Full async pipeline now, kills are instant
- **Container First-Drag Fixed.** Opening a container for the first time and dragging an item could freeze for a second.
- **Memory Bloat Fixed.** Item registry was loading all items into memory at boot and never cleaning up. Now bounded and on-demand - SORRYYYY
- **Multi-World Damage Freeze Fixed.** A race condition between realm worlds could permanently disable all damage processing on the server
- **Cross-World Combat Crash Fixed.** Players getting kicked during combat in a different world could crash the server
