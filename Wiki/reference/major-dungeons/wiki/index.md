# Overview

Major Dungeons adds new custom dungeons to the game. These dungeons are instanced and accessible through the Ancient Portal, which unlocks after reaching 100 Memories. In the dungeons, you'll encounter completely new enemies and bosses. As a reward for completing dungeons and killing the final bosses, you'll receive newly added weapons and armor, which drop at certain rates. Inspired by games such as Terraria, The Legend of Zelda, and Skyrim!

The mod is a combination of a framework that adds dungeon and RPG specific features that other modders can use to implement their own dungeons, and an asset pack that implements the content. This wiki has some [Modding Tutorials](./modder-tutorials) for those that are interested in implementing these features into their asset pack, requiring zero Java code!

# Installation

## Dependencies
- If you are running any other mods that add any HUDs to the game, it is required to use the [MultipleHUD](https://www.curseforge.com/hytale/mods/multiplehud) mod to prevent crashes. Otherwise it is not required.
- The mod is developed while balancing around the [Stunning Combat](https://www.curseforge.com/hytale/mods/stunning-combat) mod. No parry or combat mod is a requirement and you can use any that you'd like, or none at all.

## Steps
1- Download Latest from [Curseforge](https://www.curseforge.com/hytale/mods/major-dungeons/files/all?showAlphaFiles=hide)

**Server**:
2- Place downloaded file in server mods/ folder
3- Restart

**Singleplayer**:
2- Click Edit world and Open folder
3- Place downloaded file in mods/ folder
4- Go to Mods tab in-game and enable mod

# Configuration

All player data is found in your world folder at `universe/players/{UUID}.json`

The mod specific configuration data is found at `mods/MAJOR76_MajorDungeons/MajorDungeons.json`

# Start

Reach 100 Memories and craft the [Ancient Gateway](https://hytalewiki.org/w/Ancient_Gateway) using the [Arcanist's Workbench](https://hytalewiki.org/w/Arcanist%27s_Workbench).

Craft the first Fragment Key, [Fragment: Dungeon of Fear I](./fragment-dungeon-of-fear-i), and use it on the Ancient Gateway to summon a portal to the first dungeon. Make sure you are well prepared.

Defeating the boss will drop the key to the Treasure Room and provide you the first [Treasure Bag](./treasure-bag-dungeon-of-fear-i), which gives you the Fragment Key to the next dungeon.

The [Nameless God](./nameless-god) awaits you in the Treasure Room, providing you access to the mod's weapons, armor, and other items via the [Dark Coins](./dark-coin) you will loot throughout your time in the dungeon.
